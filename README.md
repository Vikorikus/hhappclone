Напрямую данные с API HeadHunter получить не удалось, так как серверы HH блокируют запросы из браузера (ошибка CORS).

Поэтому я сделал так:
Создал mock-данные (тестовый файл с вакансиями).
Сейчас приложение имитирует работу с реальным API, используя эти данные.
Всё остальное (фильтры, поиск, карточки) работает так же, как если бы ответы приходили от настоящего сервера.
